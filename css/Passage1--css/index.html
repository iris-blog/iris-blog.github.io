<选择器部分笔记>

<链接伪类选择器>

​ ★以冒号（：）开头

​ ★以下顺序不可变

​ a:link /选择未被访问的链接/

​ a:visited /选择已经被访问的链接/

​ a:hover /选择鼠标指到的链接/

​ a:active /鼠标按下未弹起时的链接/

​ ★链接样式单独指定，常用格式：①给a标签指定样式②给a链接指定样式

<:focus伪类选择器>

​ 用于选取光标所在的表单元素；一般类表单元素可获取

<复合选择器总结>

<元素显示模式>

<块元素>

​ 如：p\h1~h6\div\ul\ol\li

​ 1、特点：容器，高宽边距均可控制，自己独占一行

​ 2、注意：文字类元素不可使用块级元素，如：

标签主要存放文字，不可放div，同理h1~h6

<行内元素>

​ 如：a\strong\b\em（强调标签，斜体）\i\del（删除线）\s\ins（下划线）\u\span

​ 1、特点：高宽直接设置无效，默认宽是它本身内容的宽度，一行可以显示多个行内元素，只可以容纳文本和行内元素

​ 2、注意：链接不可嵌套

<行内块元素>

​ 1、特点：一行可以显示多个，默认宽度，高度行高边距可控

<元素显示模式转换>

​ 1、转换为块元素：display:block;

​ 2、转换为行内元素：display:inline;

​ 3、转换为行内块：display:inline-block;

<背景>

背景颜色：

background-color：一般为transparent；透明

背景图像：

background-image：none：无背景图（默认）；url：绝对or相对路径，不要加引号

背景平铺：

background-repeat ：repeat；no-repeat;repeat-x;repeat-y

背景图像固定（背景附着）：

background-attachment：scroll：背景图像随对象内容滚动；fixed：背景图像固定

背景图片位置：

background-position ：x y；背景图片位置；（xy可使用方位名词或精确单位）

○○注意：

​ ①xy都为方位名词，则与顺序无关

​ ②只有一个方名，第二个默认居中对齐

​ ③xy都为精确坐标，第一个x第二个y顺序不可变

​ ④只有一个精坐，第二个默认垂直居中

​ ⑤参数混合，第一个为x，第二个为y

★☆背景复合写法

​ background：transparent（透明） url(image.jpg)（图像） repeat-y （平铺） fixed（图像固定） top（位置）

<盒子属性>

一、边框

三种可以复合：（无顺序，单位不同）

边框样式border-style可以设置如下值：

​ none：没有边框即忽略所有边框的宽度（默认值）

​ solid：没有边框即忽略所有边框的宽度（默认值）

​ dashed：边框为虚线

​ dotted：边框为虚线
